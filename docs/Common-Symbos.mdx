# Common Symbol Entities in MDX

## Copyright and Trademark Symbols

- Copyright: &copy; (e.g., &copy; 2024 UlinkCMS)
- Registered Trademark: &reg; (e.g., &reg; Example)
- Trademark: &trade; (e.g., Example&trade;)

## Currency Symbols

- Cent: &cent;
- Pound: &pound;
- Yen: &yen;
- Euro: &euro;
- Dollar: &dollar;

## Mathematical Symbols

- Plus/Minus: &plusmn;
- Multiplication: &times;
- Division: &divide;
- Less than: &lt;
- Greater than: &gt;
- Less than or equal: &le;
- Greater than or equal: &ge;

## Arrow Symbols

- Left Arrow: &larr;
- Up Arrow: &uarr;
- Right Arrow: &rarr;
- Down Arrow: &darr;

## Quotes and Apostrophes

- Double Quote: &quot;
- Single Quote: &apos;
- Left Single Quote: &lsquo;
- Right Single Quote: &rsquo;
- Left Double Quote: &ldquo;
- Right Double Quote: &rdquo;

## Space Characters

- Non-breaking Space: &nbsp;
- Half Space: &ensp;
- Full Space: &emsp;

## Other Symbols

- Section: &sect;
- Degree: &deg;
- Ellipsis: &hellip;
- Bullet Point: &bull;
- Paragraph: &para;
